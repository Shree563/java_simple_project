Factorial Program

                
public class Factorial {
    public static void main(String[] args) {
        int number = 5; // Change this to the calculated factorial input
        int factorial = 1;
        for (int i = 1; i <= number; ++i) {
            factorial *= i;
        }
        System.out.println("Factorial of " + number + " = " + factorial);
    }
}

Explanation

This Java program calculates the factorial of a given number. It initializes a variable `number` with the input value, computes the factorial using a `for` loop, and prints the result to the console.
